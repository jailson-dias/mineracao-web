PESQUISA & DESENVOLVIMENTO

Somos uma empresa inovadora focada em Data Science

Uma das estratégias prioritárias da Keyrus é a ciência de dados, a ciência de processamento de enormes massas de dados estruturados. Big Data requer abordagens projetadas para captura, processamento, análise, categorização, armazenamento e valorização dos dados. Os métodos utilizados para transformar dados em estruturas, informações integradas e, posteriormente, em conhecimento. Assim é como a ciência de dados ou “Data Science” é comumente definida.

Com Data Science, empresas podem melhorar e obter valor real em suas operações com Big Data.

Ciência de dados para enfrentar o desafio do Big Data

Ciência dados é multidisciplinar. Isso consiste de:

  • Matemática Aplicada: em particular, as estatísticas, probabilidade, álgebra linear, etc.
  • Algorítmos avançados do tipo: Machine Learning, MapReduce, Hadoop, Spark, etc.
  • Gestão: árvore de decisão, visualização de dados etc.

Essas três disciplinas são complementares e indissociáveis. Elas formam uma convergente complexa, que pode ser definida como a ciência de grandes quantidades de dados multi-estruturados ou Ciência de Dados.

Aplicações atuais para Data Science:

A aplicação efetiva da Ciência de Dados contribui, entre outras coisas, para:

  • A exploração de metadados para extrair as informações e sua localização (análise semântica);
  • Visualização de dados;
  • Aprendizagem em grande escala de modelos tais como os modelos visuais baseados em grandes massas de dados on-line.

No entanto, qualquer incerteza ou imprecisão nos dados devem ser integrados nas várias fases do gerenciamento: abordagens por intervalos, probabilidade etc. Em grande escala, a integração contínua de dados heterogêneos, com diversas semânticas, também requer novas técnicas de classificação automatizada. Finalmente, as relações sociais entre os usuários devem ser capazes de ser bem aproveitados para melhorar a qualidade da pesquisa, em particular, com as recomendações técnicas de representação gráfica, social, etc.

Data Science para enfrentar desafios:

Para as empresas, o desafio consiste em processamento e uso dos dados disponíveis, a fim de melhorar a sua competitividade. Além de padrão, os chamados dados estruturados que já são manuseados e explorados por todos os tipos de empresas através de Business Intelligence, existe agora grandes quantidades de dados não estruturados, que vêm da Internet, mídias sociais, celulares e dispositivos inteligentes. Este é um "tsunami" de dados ou um dataclisma: mais dados foi produzido ao longo de apenas 10 anos que nunca foi produzido pela humanidade desde as suas origens.

Keyrus: Integração entre BI e Big Data

Estratégia científica da Keyrus pode ser definida como a intenção de investigar este novo campo interdisciplinar de "Ciência de dados", ou Dados estruturados. Este processo encontra-se na intersecção entre a matemática aplicada e a ciência da computação avançada. Esta especialização de investigação científica é parte do trabalho contínuo em Business Intelligence da qual Keyrus tem sido um especialista reconhecido por quase 20 anos.

Este posicionamento estratégico orienta a política de P&D do Grupo para a contribuição combinada de sua atividade original, Business Intelligence, e o advento da análise de massiva de dados estruturados, tanto a nível metodológico como a tecnologia (plataforma e serviços).

  • Inovação através de dados
  • Big Data Arquiteturas e otimização TCO
  • Consultoria Ciência dados
  • Laboratório de Big Data Agil
  • Big Data Service Factory

Cursos universitários e treinamento em Data Science

A Keyrus participa diretamente na criação de um ramo de emprego de cientistas de dados, em apoio ao Plano de Big Data do Governo. O Grupo tem, assim, cursos de Pós-graduação (Master 1, Master 2, PhD e cursos certificados) em Ciência de Dados, a fim de treinar a próxima geração de cientistas de dados para as empresas.

Curso de Ciência de Dados na École Polytechnique Engineering School

A Keyrus, em parceria com a Orange e Thales, juntou forças com a Ecole Polytechnique em 2014 para criar o curso de Ciência de Dados. Sua finalidade é apoiar programas de formação em Ciência de Dados aplicada a Big Data e, assim, criar um novo pool de recrutamento para organizações, tanto privadas como públicas.

Programa Data Science 2015

O curso também apoia cursos de formação em serviço para profissionais que trabalham para atender às necessidades crescentes de empresas para estas novas competências.

Em 2015, a Keyrus iniciou o programa de certificação junto a École Polytechnique o Data Sciences Starter Program, é direcionado para as empresas que pretendam treinar seus funcionários que lidam com Big Data.

Quem são os alvos do programa?

  • Perfis técnicos: engenheiro de dados, arquiteto de dados, gerente BDDs etc.
  • Perfis de administração / negócios, diretor comercial, marketing / CRM, consultoria de negócios etc.
  • Especialistas em inteligência de dados: estatísticos, mineradores de dados, gerente de BI etc.
