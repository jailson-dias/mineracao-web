    Descrição da Vaga

    Estágio Doutorado Pesquisa – Ciência de Dados

    O Laboratório de Pesquisas da IBM oferece vaga de estágio para estudantes de Pós-Graduação que estejam interessados em pesquisas com forte aplicação tecnológica e industrial.
    O projeto envolve a população de um banco de dados de produtos químicos com aplicação industrial através da consulta de literatura técnico-científica (livros, artigos e patentes) de forma manual ou automatizada (processamento de linguagem natural). Após preenchido, o banco de dados deverá ser consultado e seu conteúdo processado por algoritmos de aprendizado de máquina para identificar padrões e recomendar produtos químicos para uma determinada aplicação.
    Required Technical and Professional Expertise
    * Cursando Doutorado em Engenharia (Computação, Mecânica, Química, Elétrica, Eletrônica, etc.), Ciência da Computação, Química, Física ou Matemática.
    * Leitura de literatura técnico-científica e extração de dados relevantes.
    * Banco de dados relacionais: SQL, Postgres, etc.
    * Programação orientada a objetos em Python, Git e ambiente Linux.
    * Inglês intermediário.

    Preferred Technical and Professional Experience
    * Bibliotecas de Ciência de Dados: TensorFlow, Caffe(2), PyTorch, Pandas, Scikit-Learn, etc.
    * Ferramentas de DevOps: Docker, Kubernetes, Vagrant, Travis, etc.
    * Computação em nuvem: microsserviços, web APIs, CloudFoundry, etc.
