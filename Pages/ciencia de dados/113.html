    Veja onde estudar para ser um cientista de dados

    ADRIANA FONSECA
    COLABORAÇÃO PARA A FOLHA

    07/02/2016 01h00

    Não há no Brasil um curso de graduação voltado especificamente para a profissão de cientista de dados, considerada como uma das carreiras mais relevantes pelo Fórum Econômico Mundial.

    Como a análise de um grande volume de dados, ou "big data", exige técnicas estatísticas avançadas, os profissionais costumam vir de áreas com base matemática, como engenharia, estatística e ciência da computação.

    Para quem quer trabalhar na área, diz Leonardo Oliveira, cientista de dados da start-up financeira Biva, é necessário ter bom raciocínio lógico e a capacidade de analisar um projeto sob vários aspectos.

    "Quem entende o problema de uma maneira diferente usa informações pouco convencionais para aumentar o poder preditivo do seu modelo", afirma o cientista.

    Além disso, é importante gostar de trabalhar diante do computador, já que boa parte da rotina envolve olhar para os dados e gerar relatórios.

    O mais comum entre os profissionais da área é buscar uma especialização em "big data". Ou várias.

    "Para crescer na carreira é imprescindível se atualizar constantemente. Se o profissional perder contato com a inovação ficará obsoleto", diz Rodrigo Senra, 41.

    Eduardo Knapp/Folhapress
    Rodrigo Senra, 41, cientista de dados do EMC

    Ele coordena uma equipe de 16 cientistas de dados na sede carioca da multinacional EMC, que tem um centro de pesquisa na área.

    Um de seus projetos recentes foi fazer um modelo de economia de combustível para uma empresa de transporte de passageiros. "Criamos ferramentas para ajudar os gestores a otimizarem a alocação de recursos", explica.

    Para os interessados, ele recomenda procurar na internet pelos Moocs (cursos on-line abertos e massivos, em inglês), vários disponíveis de graça.
