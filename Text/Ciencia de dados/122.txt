principais bibliotecas de ciencia de dados para python r e scala by juliana monaco 04 jul 2018 big data a ciencia de dados e um campo promissor e empolgante a area de casos de uso de ciencia de dados esta em constante expansao e o kit de ferramentas para implementar essas aplicacoes esta crescendo rapidamente portanto os cientistas de dados devem estar cientes de quais sao as melhores solucoes para as tarefas especificas assim embora muitas linguagens possam ser uteis para um cientista de dados essas tres permanecem como as mais populares e sao desenvolvidas para implementar solucoes de ciencia de dados e de aprendizado de maquina neste post ha um infografico que mostra as 20 principais bibliotecas em cada linguagem de programacao que sao beneficas para os cientistas e engenheiros de dados essa selecao mostra como as linguagens se relacionam entre si e quais bibliotecas tem uma area de aplicacao semelhante embora existam muitos campos especificos de aplicacao queremos nos concentrar naqueles que sao perfeitamente adequados para aprendizado de maquina visualizacao matematica e engenharia manipulacao e analise de dados e pesquisa reproduzivel voce pode ver os campos de implementacao em cores diferentes no infografico abaixo os programas de machine learning cuidam da construcao e da implementacao dos principais algoritmos de aprendizado de maquina criando fluxos de trabalho e em geral ajudando a resolver problemas eles fornecem o kit de ferramentas principal para classificacao regressao e outros diversos problemas como parte integrante da ciencia de dados o campo da manipulacao e analise de dados representa bibliotecas que realizam a captura ingestao limpeza preprocessamento de dados e outras operacoes que permitem brincar com os dados e como resultado executar a analise em si com a ajuda de programas de visualizacao  voce pode exibir os dados visualmente o que e necessario para entendelos e interpretalos melhor esses programas contem varios graficos de visualizacao bem como diferentes opcoes de representacao as bibliotecas de matematica e engenharia fornecem as habilidades para armazenar dados numericos de uma forma conveniente e executar operacoes matematicas e calculos cientificos complexos e avancados alem disso esses programas sao usados para processar dados mais complexos como texto e conteudo finalmente os programas para pesquisa reproduzivel implementam a ideia de criar documentos que combinam codigo dados e conteudo basicamente voce pode produzir um novo trabalho do seu projeto que pode ser imediatamente publicado cada uma dessas linguagens e adequada para um tipo especifico de tarefas alem de cada desenvolvedor escolher a ferramenta mais conveniente para si r projetado principalmente para computacao
